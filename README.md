# appSIAR

## Descripción
**appSIAR** es una aplicación de seguridad avanzada desarrollada con Flutter y conectada a Firebase, diseñada para integrarse con dispositivos IoT y proporcionar funcionalidades antirrobo. La aplicación permite monitorear y controlar dispositivos de seguridad en tiempo real, recibiendo notificaciones y estadísticas para proteger mejor tu hogar o negocio.

## Características
- **Autenticación Segura:** Integración con Firebase Authentication para una autenticación segura del usuario.
- **Monitoreo en Tiempo Real:** Visualización en tiempo real del estado de los dispositivos IoT conectados.
- **Notificaciones:** Recepción de notificaciones push cuando se detectan eventos sospechosos.
- **Estadísticas:** Visualización de estadísticas y datos históricos sobre la actividad de los dispositivos.
- **Control Remoto:** Posibilidad de controlar los dispositivos IoT desde la aplicación.
